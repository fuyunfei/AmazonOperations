# 02 — 品类视图页

[← 返回索引](./index.md)

> **数据范围**：单品类内所有 ASIN，不跨品类。  
> **刷新时机**：每次上传任意报表后自动重算。

---

## 功能说明

品类视图是中间层级页面，包含 4 个 Tab，数据范围均为**当前品类内所有 ASIN**：

1. **KPI 汇总** — 品类聚合指标（GMV / 广告花费 / TACoS / ACOS）+ 各 ASIN 明细行
2. **每日告警** — 品类内所有 ASIN 的红/黄告警，按优先级排序
3. **广告优化** — 品类内所有 ASIN 的广告行动清单（P0/P1/P2）+ 内部竞争检测
4. **库存看板** — 品类内所有 ASIN 的库存状态、断货风险、补货建议

> Chat 不在品类视图内，已移至顶层导航，可跨品类提问。

---

## 输入报表

| 报表 | 来源系统 | 上传频率 | 用途 |
|------|---------|---------|------|
| 产品报表-子ASIN视图（单日快照） | Nordhive | 每日 | 筛选本品类 ASIN 的销售数据 |
| 库存报表 | Nordhive | 按需 | 筛选本品类 ASIN 的库存状态 |
| 搜索词重构 | Nordhive | 每周 | 品类内跨 ASIN 关键词重叠检测 |
| 品类映射配置 | 系统初始化 | 一次性 | 确定本品类包含哪些 ASIN |

---

## 处理的数据字段

### 来自产品报表（仅本品类 ASIN 行）

| 字段 | 处理方式 |
|------|---------|
| `gmv` | `SUM(本品类所有ASIN近7天)` → 品类总 GMV |
| `ad_spend` | `SUM(本品类所有ASIN近7天)` → 品类总广告花费 |
| `orders` | `SUM(本品类所有ASIN近7天)` → 品类总订单量 |
| `ad_spend` / ASIN | 各 ASIN 花费 ÷ 品类总花费 → 预算占比 |

### 来自搜索词重构（跨 ASIN 词重叠检测，在广告行动清单中使用）

| 字段 | 处理方式 |
|------|---------|
| `search_term` + `asin` + `match_type` | 在品类内查找：同一搜索词 × 同一投放类型 × 不同 ASIN → 标记内部竞争 |

---

## 计算逻辑

```
品类 TACoS = SUM(本品类ad_spend, 近7天)
           ÷ SUM(本品类gmv, 近7天)

品类内 ASIN 预算占比：
  ASIN_A 占比 = ASIN_A近7天ad_spend ÷ 品类总ad_spend × 100%

品类告警汇总：
  遍历本品类所有 ASIN 的告警结果
  统计红色告警数量 + 黄色告警数量
  → 显示在品类卡片入口处
```

## 品类视图与账号总览的区别

| 维度 | 账号总览 | 品类视图 |
|------|---------|---------|
| KPI 范围 | 全账号所有 ASIN | 仅本品类 ASIN |
| 展示粒度 | 品类健康摘要 | ASIN 详情卡片 |
| 跨 ASIN 分析 | 不做 | 关键词重叠检测（在广告行动清单中） |
| Chat 入口 | 无（顶层导航 Chat 可跨品类提问） | 无（顶层导航 Chat 可跨品类提问） |
